# Taylor cutouts — the emotional heart of the site

Drop one **transparent-background PNG per era** here and the app will
automatically use it for the cutout-reveal (carousel hover) and the album
hero. If a file is missing, the app gracefully shows only the album cover
— no broken image.

## Filenames (must match the album id exactly)

```
debut.png
fearless-tv.png
speak-now-tv.png
red-tv.png
1989-tv.png
reputation.png
lover.png
folklore.png
evermore.png
midnights.png
ttpd.png
showgirl.png
```

## Requirements

- **Transparent alpha channel.** Only Taylor's figure — no rectangle, no
  gradient, no backdrop. If you can see a box behind her, it's wrong.
- Portrait orientation. She now renders as the GIANT fullscreen hero
  (~70% of the viewport, rising from the era's fog behind the carousel),
  so favor tall, waist-up or full-body shots with her face in the upper
  half of the frame. The app soft-masks the bottom ~25% into the fog.
- ~1600–2200px tall, optimized PNG (or WebP renamed appropriately).
- Pick an expression per era: huge smile for Fearless/Lover, dreamy for
  Speak Now, wistful for folklore/TTPD, fierce for reputation…

## Sourcing them legally (important)

This is a public, non-commercial fan project. Publishing celebrity photos
without a license carries real legal risk (copyright + right of
publicity). In order of safety:

1. **Commission original illustrations** of Taylor in each era's style
   from an artist — you own the result. *(Strongly recommended.)*
2. Buy properly licensed press photos, then background-remove them with
   remove.bg or similar.
3. Use press photos only where you hold a valid license.

Never scrape-and-publish. Honor any takedown request (see the footer
contact). Until you add licensed/original art, the site looks complete on
album covers alone.
